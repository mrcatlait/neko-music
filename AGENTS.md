## Agent skills

### Issue tracker

Issues live as markdown files under `.scratch/<feature-slug>/`. See `docs/agents/issue-tracker.md`.

### Triage labels

Five canonical triage roles mapped to `Status:` lines in issue files; defaults match role names. See `docs/agents/triage-labels.md`.

### Domain docs

Multi-context monorepo: read `CONTEXT-MAP.md` at the repo root, then each relevant context under `docs/contexts/`; system-wide ADRs in `docs/adr/`. See `docs/agents/domain.md`.
