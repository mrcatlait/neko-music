# Copywriting Examples

Before/after pairs showing the skill's voice in practice.

## UI Buttons

| Before | After | Why |
|--------|-------|-----|
| OK | Save | Names the outcome |
| Submit | Import playlist | Verb + object |
| Yes | Delete album | States the consequence |
| Click here to import | Import | Cuts filler; button context is enough |

## Empty States

**Before:**
```
No data
Nothing here yet!
```

**After:**
```
No imports yet
Import a playlist or upload local files to get started.
[Import music]
```

## Error Messages

**Before:**
```
Error 500
Oops! Something went wrong. Please try again later.
An unexpected error occurred.
```

**After:**
```
Something went wrong
We couldn't reach the server. Check your connection and retry.
[Retry]
```

## Form Validation

**Before:**
```
Invalid input
Please enter a valid value in this field.
```

**After:**
```
Enter a URL starting with https://
```

## Headings and Descriptions

**Before:**
```
Welcome to the Import Wizard!
This powerful feature lets you seamlessly import your favorite tunes from various sources with just a few clicks!
```

**After:**
```
Import music
Add tracks from a playlist URL or local files.
```

## Feature Bullets (README / Marketing)

**Before:**
```
- Revolutionary AI-powered smart import
- Best-in-class streaming experience
- Unlimited everything!
```

**After:**
```
- Import from playlist URLs — metadata is reviewed before it enters your library
- MPEG-DASH adaptive streaming — bitrate adjusts to your connection
- Self-hosted — your files stay on your hardware
```

## Accessibility Labels

**Before:**
```
aria-label="Play button icon"
aria-label="Click to shuffle"
```

**After:**
```
aria-label="Play"
aria-label="Shuffle"
```

Role is announced by the screen reader; visible context is not repeated.

## Material Design Word Choice

### Common words over jargon

| Before | After | Why |
|--------|-------|-----|
| Enable location history | Turn on location history | Common word (Material) |
| Buffering… | Preparing video… | Avoid industry jargon |

### Consistent verbs

| Before | After | Why |
|--------|-------|-----|
| Delete photo / Remove track | Remove photo / Remove track | Same verb for same action |
| Remove photo from library page? | Remove photo? | Shorter dialog title |

### Present tense

| Before | After | Why |
|--------|-------|-----|
| Message has been sent | Message sent | Present tense for product state |
| Changes will be saved | Save changes? | Active, direct |

### Pronouns

| Before | After | Why |
|--------|-------|-----|
| We'll show you popular albums | Get started with popular albums | Avoid `we`; focus on user |
| Change your settings in My Account | Change your settings in Account | Don't mix `your` and `my` |
| I agree to the terms of service | *(keep as-is)* | First person OK for legal ownership |

### UI element references

| Before | After | Why |
|--------|-------|-----|
| Click the Continue button | Click Continue | Refer by label, not control type |
| Press the menu item labeled Import | Select Import | Label, not element type |

### Objective-first structure

| Before | After | Why |
|--------|-------|-----|
| Drag a track to the trash to remove it | To remove a track, drag it to the trash | Lead with objective (Material) |

### Absolutes

| Before | After | Why |
|--------|-------|-----|
| We'll never share your data | Your data isn't shared | Factual, not absolute promise |
