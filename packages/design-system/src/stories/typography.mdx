# Typography

Our typography system is built on a consistent scale and provides a comprehensive set of text styles for various use cases. The system uses Noto Sans as the primary font family, with fallbacks to Roboto, Helvetica, and Arial.

## Font Family

```css
font-family: 'Noto Sans', 'Roboto', 'Helvetica', 'Arial', sans-serif;
```

### Truncate
```html

```

### Ellipsis
```html

```

## Best Practices

1. Use display styles sparingly and only for the most important headlines
2. Maintain hierarchy by using appropriate styles for different levels of content
3. Use body styles for main content to ensure readability
4. Apply label styles for form elements and interactive components
5. Use utility classes like `truncate` and `ellipsis` for handling text overflow
6. Keep line lengths comfortable for reading (ideally 45-75 characters)
7. Ensure sufficient contrast between text and background colors
